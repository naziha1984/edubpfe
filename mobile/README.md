# EduBridge Mobile

Flutter mobile application for EduBridge.

## Prerequisites

- Flutter SDK (>=3.0.0)
- Dart SDK
- Android Studio / Xcode (for mobile development)

## Getting Started

1. Install dependencies:
```bash
flutter pub get
```

2. Run the app:
```bash
flutter run
```

## Build Commands

- **Android APK**: `flutter build apk`
- **Android App Bundle**: `flutter build appbundle`
- **iOS**: `flutter build ios`
